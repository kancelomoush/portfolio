Data Warehouse & BI Solution for Distrisys

This project involved the end-to-end implementation of a business intelligence solution for the Distrisys data warehouse. The primary goal was to build a robust, high-performance decision-making model, from data structuring to final analysis.

The process began with modeling the data warehouse using a star schema, defining fact tables and dimensions in SQL Server. An ETL (Extract, Transform, Load) pipeline was developed with SQL Server Integration Services (SSIS) to populate the warehouse.

Subsequently, a complete OLAP cube was built using SQL Server Analysis Services (SSAS), enabling multidimensional analysis. A tabular model was also developed, enriched with a semantic layer including calculated columns, DAX measures, KPIs, and hierarchies to improve usability. Security was a key component, with roles implemented to control data access based on user profiles. The final model is structured, secure, and ready to be used for effective business analysis.

Key Features & Technologies

Data Warehouse Modeling

Star schema design and implementation in SQL Server.

ETL Process

Data integration and transformation using SQL Server Integration Services (SSIS).

OLAP Cube

Multidimensional model built with SQL Server Analysis Services (SSAS).

Tabular Model

Semantic layer with DAX measures, calculated columns, and KPIs.

Security

Implementation of role-based security to restrict data access.
