Project Overview: Detecting Toxic Comments with NLP (LSTM)

This project is a Flask-based web application for detecting toxic comments using a Bidirectional LSTM model. It showcases skills in text classification, sentiment analysis, and model deployment. This was the second task of my data science internship at Intern Intelligence.

Key Features & Technologies

Data Processing

Cleaned and vectorized text data using TensorFlow’s TextVectorization.

Model Training

Implemented Bidirectional LSTM with word embeddings for better text representation.

Evaluation

Measured performance with accuracy, precision, recall, and F1-score.

Deployment

Integrated the model into a Flask application for real-time toxicity detection.

Tech Stack

  • TensorFlow & Keras
  • Flask
  • SpaCy & NLTK
  • LSTM
