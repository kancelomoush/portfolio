Project Information

  • Category: Data Engineering, Cloud
  • Project date: 2025
  • Project URL: github.com/moussalasfar/airflow-github-aws-pipeline

Project Overview

Abstract: This project is an end-to-end data engineering pipeline using Apache Airflow, AWS, and the GitHub API. It automates extraction of repository metadata, processes and cleans the data, and stores it securely in AWS S3, showcasing cloud orchestration and API integration skills.

Key Features & Technologies

GitHub API Integration

Automated extraction of repository metadata for multiple users using secure token-based authentication.

Apache Airflow Orchestration

Task scheduling and automation with DAGs for extraction, processing, and uploading to S3.

AWS S3 Data Storage

Secure storage of processed data in S3 buckets with proper IAM permissions.

Technologies Used

  • Apache Airflow
  • AWS S3 & EC2
  • GitHub API
  • Python
  • Pandas
  • Ubuntu

Email Notifications

Sends an email to administrators with the detection details.

MongoDB Integration

Stores information about detected individuals (including their image).

High Confidence Threshold

Reduces false positives through a configurable confidence threshold.
