Project Overview: Custom Debian Live Distro

This project demonstrates how to build a customized, bootable Live Debian distribution from scratch. The resulting OS image supports both legacy BIOS and modern UEFI systems, making it highly portable. The primary goal is to create a Debian environment with pre-installed packages, configurations, and certificates, ready for deployment.

Key Features & Technologies

Debootstrap

Used to create a minimal, foundational Debian "bullseye" system in a local directory.

Chroot Environment

Entered the bootstrapped system to install a kernel, the `live-boot` package, and custom software.

SquashFS

The entire configured root filesystem was compressed into a `filesystem.squashfs` file for the live environment.

Hybrid Boot (BIOS/UEFI)

Configured GRUB2 and `xorriso` to create a single `.iso` file that is bootable on both BIOS and UEFI machines.

Tech Stack

  • Debian "bullseye"
  • GRUB2
  • `debootstrap`, `xorriso`
  • Bash Scripting
